# DTO(Data Trancfer Object)
: 데이터 전송을 목적으로 만들어지는 객체(class)
  주로 원격 호출(RESTful API) 간의 데이터를 전달하기 위해 사용
  주로 외부 시스템과 상호 작용할 때 데이터를 전달하고 전송하는데 사용
  데이터 전송을 위한 필드만 포함하고, 비즈니스 로직을 가지지 않는 경우가 많음


# VO (Value Object)
: 변하지 않은 값을 담아내는 객체(클라스)
  읽기 전용 속성
  비즈니스 도메인에서 중요한 역할을 함. 비즈니스 규칙을 나타내기 위해 사용
  모델에서 일부만 사용이 되고, 특정 클래스의 상태를 나타내거나 변하지 않는 객체를 생성
  
  
*RESTful API 이란?
	웹(http) 통신을 위한 스타일 중 하나
